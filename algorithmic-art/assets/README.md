# Assets Directory

This directory contains templates, color palettes, and example outputs for the algorithmic-art skill.

## Contents

### Color Palettes
- Example color schemes for generative art
- Gradient definitions
- Palette JSON files

### Templates
- SVG templates for vector output
- Configuration files for common patterns

### Example Outputs
- Sample generated artworks
- Reference images showing different techniques
- Before/after examples

## Usage

Assets in this directory can be referenced by the algorithmic art generation scripts or used as inspiration for new generative designs.
